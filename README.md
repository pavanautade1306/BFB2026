# Crop Decision Intelligence System

Demo web app that recommends crops based on environmental and market factors.

How to run
- Open `INDEX.html` in your browser (double-click or use a local server).
- Enter soil pH, soil type, rainfall, temperature, and irrigation availability.
- Adjust "Market Price Importance" to prioritize market-driven choices.

What it includes
- `INDEX.html` — web UI
- `script.js` — sample dataset and recommendation engine (rule-based)
- `styles.css` — layout + visualization

Notes
- This demo uses a simple scoring algorithm for demonstration. For production, integrate real sensor data, weather APIs, and a trained model.
